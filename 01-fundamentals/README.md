# Fundamentals of Programming

##  Overview
This course covers the essential fundamentals that every programmer must understand before starting coding.

It explains how computers work internally and introduces the core concepts that build a strong foundation for learning programming.


##  Topics Covered
- How computers work
- Binary number system
- Basic computing concepts
- Core programming fundamentals


## 📂 Course Content
This course is mainly **theoretical** and does not include practical coding exercises.

All materials are organized as PDF files:
- 17 PDF files containing explanations and notes
