# Introduction to Programming with C++ - Level 1

##  Overview
This course is the first step into actual programming using C++.

After building a strong foundation in problem solving and algorithms, this course focuses on transforming logical thinking into real working code.


##  Topics Covered
- Introduction to C++ language
- Variables and data types
- Input / Output operations
- Conditions (if / else)
- Loops (for / while)
- Functions basics
- Problem solving using C++


##  Course Content
- Implementation of problems using C++
- Converting flowchart-based solutions into actual code
- Practical exercises to reinforce programming concepts

This course represents the transition from **theoretical problem solving** to **real coding**.


##  Learning Approach
The course is designed to:
- Translate problem-solving skills into code
- Understand how programming languages work in practice
- Build coding habits step by step
- Focus on logic first, then implementation


##  Connection to Previous Levels
- In previous courses: problems were solved using flowcharts (without code)
- In this course: the same thinking is translated into actual C++ code

This ensures a smooth transition from logic to implementation.


##  Connection to Next Levels
- The same problems introduced earlier are reused again
- In future courses, they will be solved using:
  - More advanced concepts (like functions and modular design)
  - More professional and optimized approaches

This repetition builds deep understanding and strong problem-solving skills.


##  What I Learned
- How to write my first real programs using C++
- How to convert logic into code
- Understanding syntax and structure of a programming language
- The importance of clean and organized code
- That coding is just a tool to implement logical thinking
