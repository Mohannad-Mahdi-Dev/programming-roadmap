# Algorithms & Problem Solving -(02-algorithms-level-1)

##  Overview
This course introduces the fundamental concepts of algorithms and problem solving.

The focus is on building strong logical thinking skills before writing any code. It emphasizes understanding how to approach problems step by step using structured thinking.


##  Topics Covered
- What are algorithms
- Problem solving techniques
- Flowchart fundamentals
- Step-by-step logical thinking
- Breaking down problems into smaller parts


## 📂 Course Content
- 50 problems organized in PDF format
- Each problem includes:
  - Problem description
  - Solution using flowcharts (no code)

All solutions are implemented using **flowcharts only**, without writing any programming code.


##  Learning Approach
This course is designed to train the mind to think like a programmer before jumping into coding.

Instead of focusing on syntax, the focus is on:
- Understanding the problem
- Designing the solution
- Visualizing the logic using flowcharts

---

##  Connection to Next Courses
- The same 50 problems will be reused in the next levels:
  - In the next course, these problems are solved using programming concepts such as functions and other core topics.
  - In later stages, the same problems are solved using more advanced and professional approaches similar to real-world and company-level solutions.

This approach reinforces learning by solving the same problem in multiple ways and levels of complexity.
